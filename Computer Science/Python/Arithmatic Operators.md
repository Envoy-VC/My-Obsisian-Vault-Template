# Arithmetic Operators

From **Highest** to **Lowest** precedence:

| Operators | Operation | Example         |
| --------- | ----------------- | --------------- |
| \*\*      | Exponent          | `2 ** 3 = 8`    |
| %         | Modulus/Remainder | `22 % 8 = 6`    |
| //        | Integer division  | `22 // 8 = 2`   |
| /         | Division          | `22 / 8 = 2.75` |
| \*        | Multiplication    | `3 * 3 = 9`     |
| -         | Subtraction       | `5 - 2 = 3`     |
| +         | Addition          | `2 + 2 = 4`     |

  
Examples of expressions in the interactive shell:

```python

>>> 2 + 3 * 6

20

```

```python

>>> (2 + 3) * 6

30

```

```python

>>> 2 ** 8

256

```

```python

>>> 23 // 7

3

```

```python

>>> 23 % 7

2

```

```python

>>> (5 - 1) * ((7 + 1) / (3 - 1))

16.0

```
